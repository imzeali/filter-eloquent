# filter-eloquent
Quickly filter data by concatenating query strings
### Install

```
composer install imzeali/filter-eloquent
```
### Supported operators
```
  ' '           // equal to
  'eq'          // equal to
  'ne'          // not equal to
  'gt'          // Is greater than
  'ge'          // great than and equal to
  'lt'          // less than
  'le'          // less than and equal to 
  'like'        // LIKE
  'in'          // IN
  'not_in'      // NOT IN
```
